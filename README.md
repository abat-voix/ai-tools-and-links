# AI Tools And Links

Репозиторий с полезными материалами и учебными примерами к курсу «AI-разработчик: от API до агентов».

Здесь собраны:

- стартовые инструменты для установки;
- ссылки на API, платформы, токенайзеры, IDE и AI-сервисы;
- небольшие Python-примеры, которые можно разбирать на занятиях или использовать как основу для своих экспериментов.

## Для чего нужен этот проект

Этот репозиторий решает сразу две задачи:

1. Дает одну точку входа для студентов курса.
Здесь можно быстро понять, что установить, где зарегистрироваться и какие сервисы пригодятся для практики.

2. Служит мини-набором учебных примеров.
В проекте есть скрипты по темам работы с LLM: дистилляция, подсчет токенов, prompt caching, routing и расчет стоимости чат-бота.

### Что находится в корне

- `README.md` - краткое описание репозитория, его назначения и структуры.
- `tools_and_links.md` - основной файл с полезными ссылками и сервисами для курса.
- `pyproject.toml` - зависимости и метаданные Python-проекта.
- `poetry.lock` - зафиксированные версии зависимостей.

### Что находится в `m1-b2`

Папка `m1-b2` содержит учебные примеры по теме "Оптимизация и экономика ИИ":

- `1_distillation.py` - пример идеи дистилляции через получение вероятностей ответов teacher-модели.
- `1_distillation_groq.py` - похожий пример с использованием Groq.
- `2_quantization.py` - демонстрационный пример по теме квантования.
- `3_count_tokens.py` - пример подсчета токенов.
- `4_chat_bot_calc.py` - расчет примерной стоимости чат-бота при разных провайдерах и сценариях нагрузки.
- `5_routing.py` - пример routing-подхода для распределения запросов.
- `6_prompt_caching.py` - демонстрация prompt caching на повторяющихся запросах.
