# Шпаргалка по CLI-командам Ollama

## `ollama pull`

Скачивает модель из реестра `ollama.com/library`. Аналог `docker pull`.

```bash
# По умолчанию скачивает версию latest
ollama pull qwen3:8b

# Конкретная квантизация
ollama pull qwen3:8b-q8_0
```

## `ollama run`

Скачивает модель, если ее еще нет локально, затем запускает ее и открывает чат.
Это самая частая команда в работе.

```bash
ollama run qwen3:8b
```

Для выхода:

```bash
Ctrl+D
```

или

```bash
/bye
```

## `ollama list`

Показывает все модели, которые уже скачаны на диск.
Полезно для контроля места: модели могут занимать десятки гигабайт.

```bash
ollama list
```

Если нужно освободить место, используйте:

```bash
ollama rm qwen3:8b
```

## `ollama show`

Показывает информацию о модели: архитектуру, количество параметров, квантизацию, размер контекстного окна.

```bash
ollama show qwen3:8b
```

Это удобно для проверки:
"Какая именно версия модели у меня установлена?"

## `ollama cp`

Копирует модель под новым именем.
Обычно используется вместе с `Modelfile`, когда вы создаете кастомную версию модели с другими параметрами.

```bash
ollama cp qwen3:8b my-qwen3:demo
```

## `ollama launch vscode`

Команда для подключения локальной модели к `VS Code Copilot Chat`.
Модель может запускать команды терминала, редактировать файлы и исправлять ошибки.

```bash
ollama launch vscode
```

## Полезные алиасы

Добавьте в `.zshrc` или `.bashrc`:

```bash
alias orun="ollama run"
alias olist="ollama list"
```
